# Project Overview

Plain HTML/CSS/JavaScript web application.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (no frameworks)

## Code Style

- Use semantic HTML elements
- Keep CSS in separate `.css` files
- Keep JavaScript in separate `.js` files
- Use meaningful class names (BEM or similar)
- Indent with 2 spaces

## File Structure

```
/
├── index.html
├── css/
│   └── styles.css
└── js/
    └── main.js
```

## Guidelines

- Prioritize accessibility (ARIA labels, semantic markup)
- Mobile-first responsive design
- No external dependencies unless necessary
